# Projeto Banco Java

Projeto desenvolvido para estudos da linguagem Java, com foco na aplicação dos conceitos fundamentais de programação orientada a objetos.

## Sobre o projeto

O sistema simula operações básicas de um banco, permitindo que usuários realizem ações como:

* Depósito
* Saque
* Transferência entre contas
* Consulta de saldo

Toda a interação com o sistema é realizada através do terminal, sem interface gráfica.

## Objetivos

Este projeto tem como objetivo praticar conceitos essenciais da linguagem Java, como:

* Classes e objetos
* Encapsulamento
* Métodos
* Construtores
* Coleções
* Tratamento de exceções
* Organização de projetos

## Tecnologias utilizadas

* Java
